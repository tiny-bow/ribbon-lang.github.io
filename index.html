Ribbon

An embeddable programming language offering a new approach to realtime, data-oriented systems

Ribbon utilizes the latest research in computer science to solve its oldest problem:
Software platforms are always growing.

With Ribbon, you can easily ensure performance, safety and total control; enabling real-time, deeply extensible and sustainable software. 

import std.Console

using Exception

Exception = effect a =>
    throw : a -> !

checked-div = fun a, b =>
    if b == 0
        then throw "Division by zero"
        else a / b

handled-div = fun a, b =>
    with Exception _
        throw msg =>
            Console.log msg
            cancel 0
    checked-div a, b

From the beginning of extensible software there has always been a conflict, between the interests of the developers of software platforms (whom we will call hosts), and the interests of the developers of extensions to those platforms (whom we will call guests). This is simply the nature of the beast, it seemed. Hosts want to make their platforms secure, guests want flexibility. Hosts want to make their platforms fast, guests want to write extensions quickly without worrying about low-level details. Ribbon, however, presents a new approach to this problem. We perform deep analysis of the guest code, using the latest research in type theory and emerging patterns from the industry, allowing us to ensure it is both fast and does what it says it does. This allows us to unite hosts and guests, in their shared interest of fast, safe and extensible software.
